Web-Marketing Tools for WordPress

== Description ==

This plugin gives your site some functions for web-marketing, for example, making invitations on Facebook, and so on.

== Installation ==

1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place `<?php do_action('plugin_name_hook'); ?>` in your templates

= 1.0  =

Release!

= 1.1.0  =

New version Izanai makes it possible to control the design of Facebook-Login-Button.
In particular, you can control...
  -- font-size of the message on the button
  -- margin of the button
  -- corner-roundness of the button
  -- shadow size of the button
  -- font-color in mouse-off, mouse-over and mouse-click
  -- button-color in mouse-off, mouse-over and mouse-click

= 1.1.1  =

By using short-code [izanai-button-fb], new version Izanai makes it possible to set the Facebook-Login-Button anywhere in your page or post.

= 1.1.2  =

New version Izanai makes it possible to decide whether you get facebook-user's birthday and location or not.
You can decide whether or not on "izanai-settings".

= 1.1.3  =

New version Izanai's comment of plugin-manager in dashboard is updated for internationalization.
